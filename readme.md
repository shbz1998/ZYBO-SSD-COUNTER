# ZYBO SEVEN SEGMENT DISPLAY COUNTER
## SHAHBAZ HASSAN KHAN MALIK

Requirement: Zybo Z7-10 FPGA, Vitis Suite

Functionality: Use push button to pause/resume/reset the counter. It counts from 0 to 99 and rolls over.
